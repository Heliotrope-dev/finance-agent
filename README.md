# Invest Agent

多市场（A股 / 港股 / 美股）行情查询 + 财务数据 + 新闻资讯交叉验证工具，AI 只做交叉核实和总结，不做黑箱荐股。

线上地址：[invest.heliotrope.online](https://invest.heliotrope.online)

## 设计原则

- **只摆事实，先讲依据**：详情页先看一手资讯（官方公告/富途资讯/财新摘要，各市场优先级不同）和原始数据（K线、财务表、统计指标），AI 分析摆在后面，交叉核实新闻、财务、技术面是否一致，不直接给"买入/卖出"这类指令。
- **技术面信号本地算，不靠 AI 编**：均线金叉死叉、MACD、盘中实时动量这些都是 pandas 本地算好喂给 AI 的，AI 只负责核对是否跟消息面一致，避免"技术面尚可"这种空话。
- **诚实的追踪记录**：每次生成交叉验证分析会记录当时价格和 AI 判断的方向倾向，满 7 天自动补录价格算一个方向一致率——这是历史记录的客观统计，不是"AI 荐股胜率"，页面上会明确标注不构成投资建议。

## 功能

- **多市场行情**：A 股按涨跌幅展示，港股按东财人气榜排热度，美股用固定核心股名单，都支持点击卡片直接跳转详情页。
- **实时价格**：详情页价格每 15 秒自动刷新（`st.fragment`），有真实数据变化时会有红涨绿跌的闪烁提示。
- **分时图**：A 股用新浪分钟数据，港股/美股优先走本地 Futu OpenD 网关的真实分时推送，没有 Futu 连接时退化成日 K。
- **AI 深度分析**：资讯解读、财务摘要、对比大盘、技术面与消息面交叉验证、总结性分析（带 0-100 综合评分），打开详情页自动生成，流式（打字机效果）显示，生成后缓存，不会重复调用。
- **指数详情页**：K 线/分时图 + 成分股（按涨跌幅排序的市场强势股，不是严格意义的官方成分股名单）+ AI 分析。
- **热门板块**：行情页每个市场下方展示成交额最高的 9 个行业板块（红涨绿跌），"更多板块"展开到前 30；A 股走同花顺行业板块接口，港股/美股走 Futu 板块快照，用成交额代理热度，不是官方板块人气榜。
- **自选股**：搜索添加时自动判断股票在哪个市场（"苹果"只在美股上市就直接添加，"阿里巴巴"港股美股都有就弹出来问），跨市场统一管理，列表按涨跌幅色块展示。
- **登录/账号**：邮箱+密码登录，跟 math-agent 共用同一套 Supabase 账号体系。

## 技术栈

- 数据源：[AkShare](https://akshare.akfamily.xyz)（免费开源）+ [BaoStock](http://baostock.com)（A 股主数据源，稳定免注册）+ 新浪财经（实时行情/分时）+ 东方财富公告中心（A 股官方公告，永久免费）+ 财新（资讯兜底）+ [Futu OpenAPI](https://openapi.futunn.com)（港股/美股实时行情、分时、资讯搜索，需要本地/服务器跑 OpenD 网关）
- 分析：DeepSeek（`deepseek-v4-flash`），流式输出
- 前端：Streamlit
- 账号：Supabase
- 追踪记录：SQLite（`data/track_record.db`，不入库）

## 本地运行

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
# 在 .streamlit/secrets.toml 填入 DEEPSEEK_API_KEY / SUPABASE_URL / SUPABASE_KEY
streamlit run app.py
```

港股/美股的实时行情、分时图、资讯搜索需要本机或服务器另外跑一个 [Futu OpenD](https://openapi.futunn.com/futu-api-doc/quick/opend-forward.html) 网关（监听 `127.0.0.1:11111`），没有这个连接时会自动退化成延迟行情/日 K/财新新闻兜底，不影响基本可用性。

## 已知限制

- 东方财富接口对高频请求有临时限流，`data_sources.py` 里对东财相关调用加了请求间隔和重试（`_with_retry(..., throttle=True)`），其它数据源（新浪/BaoStock/财新）不受这个限流影响。
- 东财的人气榜等个别接口偶尔会返回空响应，已经加了兜底降级，不会把原始异常甩给用户。
- akshare 内部部分接口用 py_mini_racer（V8 引擎）做反爬 JS 执行，不是线程安全的——不能并发调用，`get_multi_index_snapshot` 等多指数场景是特意串行执行的。
- 指数"成分股"板块不是严格的官方成分股名单（A 股宽基指数成分股太多没法全量拉实时行情，港股/美股没找到带股票代码的免费官方成分股源），是"这个市场当前涨幅最大的一批股票"，页面上有如实标注。
- 港股/美股实时行情、分时图、热门板块等依赖本地 Futu OpenD 网关，进程每次重启后第一次建立连接实测要几十秒到几分钟不等（Futu 自己的登录握手延迟，不是本项目代码能控制的），这段时间内相关功能会先降级显示"暂不可用"，连接成功后自动恢复，不需要手动干预。
- 只支持股票和指数，不支持基金/债券/期货/期权。
